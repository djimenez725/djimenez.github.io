Healing Touch

Trabajo en equipo en el que se creó una página web para una masajista. El objetivo principal fue diseñar una plataforma fácil de usar para los usuarios, pero que a la vez reflejara una estética natural y relajante. Me encargué de gestionar las imágenes y el diseño visual de la página, siempre alineándome con las preferencias y necesidades de la clienta.

Pawsitive RescYou

Desarrollé una página web para mi proyecto de emprendimiento, el cual tengo la intención de seguir expandiendo a largo plazo. Este proyecto tiene como objetivo reducir la cantidad de animales abandonados en Puerto Rico, rescatándolos y, al mismo tiempo, ofreciendo a personas que necesiten de su amor y compañía. La página está diseñada para invitar a los usuarios a descargar la aplicación Pawsitive RescYou. Fue creada desde cero utilizando el script language HTML y CSS.
